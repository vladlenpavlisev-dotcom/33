# Moodboard: Империя Чувств

- Цвета: белый, кремовый, светло-бежевый, золотистый, зелёный (натуральные оттенки)
- Материалы: шелк, лен, дерево, стекло, камень
- Атмосфера: мягкий рассеянный свет, свечи, ароматы масел, минимализм
- Вдохновение: восточные SPA, природа, дзен, гармония
- Визуальные референсы: павильон на берегу, хаммам, йога, чайная церемония, детали (руки, вода, цветы)

_Для визуализации moodboard используйте Pinterest/Canva, добавьте 10-15 референсов по этим критериям._
